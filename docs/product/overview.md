# Product Overview

## Goal

Build a basic church website that presents:

- the history of the church,
- renovation information,
- practical information (including events).

## Audience

- Visitors and parishioners looking for practical details.
- People interested in historical context and restoration progress.

## Scope (Current)

- Informational website (no CMS for now).
- Content stored locally in the codebase.
- Internationalized from day one with `next-intl`.

## Locales

- `nl` (default)
- `fr`
- `en`

## Non-goals (for now)

- CMS integration.
- Remote content fetching for core page content.
