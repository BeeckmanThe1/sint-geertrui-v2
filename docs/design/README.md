# Design References

Use this folder for design links and implementation notes.

## Suggested files

- `figma-links.md`: canonical links to screens/components.
- `tokens.md`: spacing, typography, and color notes.
- `content-guidelines.md`: tone/voice and copy constraints.

## Conventions

- Put date and owner for each design update.
- Link every page spec in `docs/pages/` to a design source.
- Capture implementation-relevant notes (breakpoints, spacing, interactions).
